TexasPrimary2016: Update

Nov 21, 2016

For the past six months I have not worked much on my TexasPrimary2016 project. Recently, since successfully submitting ggloop to CRAN, I have picked back up this project.

So far, I have cleaned up the code - better variable names, placing common objects in lists, using better map colors, etc - and renamed some of the files I have been working on. I am planning to use the actual 2016 election data within the project, so I am considering a new name for the project.
